# G.I.D.E. (Genshin Impact Damage Emulator)

読み方は「ジード」

## はじめに

作業環境はVSCodeを使用し、Node.js/yarnをインストールしておいてください。

## ビルド

1. リポジトリをクローン
1. 作業ディレクトリに移動
1. 依存パッケージをインストール
1. デバッグ環境でビルド・実行

```
$ git clone https://github.com/PGseatake/GenshinImpactDamageEmulator.git
$ cd GenshinImpactDamageEmulator\src
$ yarn install
$ yarn dev
```
